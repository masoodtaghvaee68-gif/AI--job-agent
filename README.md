# AI Job Agent (Email → Analysis → Telegram)

An AI-powered assistant that helps me understand job opportunities deeply.

## What it does
- Reads job-related emails (LinkedIn, StepStone, etc.)
- Extracts job information
- Translates content into Persian
- Explains abbreviations (IT / Cloud / HR terms)
- Analyzes requirements and needed background knowledge
- Creates a structured job brief
- Sends the final report to Telegram

## Why this project
I am transitioning into IT / Cloud and want to:
- Learn job market requirements
- Improve my technical and professional understanding
- Automate repetitive analysis tasks using AI

## Tech Stack (MVP)
- n8n (workflow automation)
- OpenAI (analysis, translation, explanation)
- Gmail (job email source)
- Telegram Bot (final delivery)

- [ ] Gmail trigger for job-related emails
